# 大学生就业信息管理系统 python1083

## 项目概述

大学生就业信息管理系统是基于Python开发的一套在线平台，旨在为高校学生、企业和管理员提供便捷的就业信息管理服务。

## 技术栈

- **后端语言**：Python 3.7
- **数据库**：MySQL
- **前端技术**：未具体说明
- **开发工具**：PyCharm

## 功能模块

### 前台功能

- 企业注册
- 学生注册
- 在线留言板
- 企业查看

### 后台功能

- 管理员信息管理
  - 企业注册管理
  - 学生注册管理
  - 留言板管理
  - 就业信息统计
    - 添加学生就业情况
    - 就业与未就业统计（饼状图显示）

### 企业功能

- 个人资料修改
- 招聘岗位发布
- 求职应聘审核
  - 查看学生简历

### 学生功能

- 个人资料修改
- 查看我的求职情况

## 系统角色

- 企业
- 管理员
